# 贊助須知

在連結內填寫` 遊戲內 ID `與` 贊助金額 `後結帳

煩請務必填寫` Minecraft ID `，倘若` ID `為` 空 `或` 不存在 ` ，將視為無償贊助。

(基岩版玩家 請在` Minecraft ID `前加上半形句號` . `)


贊助為` 累積制 `

倘若您第一次贊助` 200 `、第二次贊助` 270 `，您將可以獲得` 470 `的福利。


伺服器方不會以強制、或任何威脅性質的行為要求玩家贊助本伺服器
