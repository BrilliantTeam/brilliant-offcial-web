## 物品堆疊
以下物品可以被 <span class="label label-info">堆疊至 64 個</span> ：

<table>
  <tr>
    <td>各種山羊角</td>
    <td>各種噴濺藥水</td>
    <td>各種滯留藥水</td>
  </tr>
  <tr>
    <td>各種礦車</td>
    <td>各種船</td>
    <td>各種床</td>
  </tr>
  <tr>
    <td>各種界伏盒</td>
    <td>各種馬鎧</td>
    <td>各種桶子（除生物桶）</td>
  </tr>
  <tr>
    <td>各種唱片</td>
    <td>各種湯品</td>
    <td>各種旗幟</td>
  </tr>
  <tr>
    <td>各種旗幟圖案</td>
    <td>雪球</td>
    <td>蛋糕</td>
  </tr>
  <tr>
    <td>馬鞍</td>
    <td>終界珍珠</td>
    <td>盔甲座</td>
  </tr>
  <tr>
    <td>狼鎧</td>
    <td>各種告示牌</td>
    <td>各種懸掛式告示牌</td>
  </tr>
</table>

而被 <span class="label label-info">堆疊至 64 個</span> 的物品，
無法在 <span class="label label-info">鐵砧</span> 中被操作。

{% note danger %}
部分【堆疊物品】經 快速使用 後，
有可能導致 堆疊物品的遺失情況 ，
此類似情況則不予補償處理，
請玩家正常的慢慢使用。

目前已知有此問題的【堆疊物品】：
- 水桶、岩漿桶、粉雪桶
{% endnote %}
{% note warning %}
帶有大量 NBT 的堆疊物品（例如 64 個裝滿物品的界伏盒），
在放入容器時，玩家可以發現到數量變回一個，
這是正常的情況，不是伺服器吃掉你的東西，也不是模組問題，

這是為了避免玩家因巨量封包問題，
導致 BAN 人容器的產生，所進行的資料操作，
玩家不需擔心因這情況導致的物資遺失。
{% endnote %}
