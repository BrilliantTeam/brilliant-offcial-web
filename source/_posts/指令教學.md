### <參數> 有哪些種類？

-   <span class="label label-info">player:[玩家名]  <span class="label label-info">- 查找這位玩家的紀錄
-   <span class="label label-info">uuid:[玩家UUID]  <span class="label label-info">- 同上，但一般玩家通常看不到其他玩家的UUID
-   <span class="label label-info">startDate:[年-月-日]  <span class="label label-info">- 從什麼時間以後的紀錄（大範圍）
-   <span class="label label-info">ednDate:[年-月-日]  <span class="label label-info">- 紀錄會截至到什麼時間（大範圍）
-   <span class="label label-info">startTime:[時:分:秒]  <span class="label label-info">- 從什麼時間以後的紀錄（小範圍）
-   <span class="label label-info">ednTime:[時:分:秒]  <span class="label label-info">- 紀錄會截至到什麼時間（小範圍）
-   <span class="label label-info">...每個紀錄也都會有專屬的個別參數 </span> 

<br />

而每一個不同的紀錄也會有個別不同的參數，在這裡不多做教學，
因為太多了.... 您可以透過 <span class="label label-info">不輸入參數</span> 的方式來查看那些 <span class="label label-info">特定參數</span> ，
系統訊息會告知您這個類別的紀錄中，可以使用的參數。

<br />

而您也可以透過 <span class="label label-info">疊加多個參數</span> 來達成細節查詢的效果，
例如： <span class="label label-info">/shoprecordtr player:RiceChen_ action:ORIGINAL startDate:2022-09-05</span> ，
就可以查到 <span class="label label-info">RiceChen_</span> 在 <span class="label label-info">原價交易</span> 內於 <span class="label label-info">2022年9月5日</span> 後的 <span class="label label-info">商品交易紀錄</span> ，
而 <span class="label label-info">action:ORIGINAL</span> 就是 <span class="label label-info">商品交易紀錄</span> 中的一個 <span class="label label-info">專屬個別參數</span> 。

# 每日簽到
在本服，你可以透過 <span class="label label-info">/bsign</span> 來進行每日簽到，
每日簽到可以獲取少許的 <a href="https://www.brilliantw.net/%E6%8C%87%E4%BB%A4%E6%95%99%E5%AD%B8#%E8%BD%89%E6%8F%9B%E7%82%BA%E4%B9%99%E5%A4%AA">乙太</a> 與 <span class="label label-info">隨機數煙火</span> 獎勵，
倘若您是 <span class="label label-info">當日第一位</span> 簽到的玩家的話，則可以獲取 <span class="label label-info">雙倍</span> 獎勵，
而連續簽到達 <span class="label label-info">特定天數</span> 後，則將額外獲取 <a href="https://www.brilliantw.net/%E8%B4%8A%E5%8A%A9%E8%B3%87%E8%A8%8A/#%E8%B4%8A%E5%8A%A9%E7%A6%8F%E5%88%A9">水芥</a> 與 <span class="label label-info">鞘翅</span> 獎勵！

# 方塊查詢

           1. 輸入 /co i 開啟查詢模式

           2. 點按您想查詢的方塊，會跳出系統訊息，可以看到 時間/玩家/行為

           3. 再次輸入 /co i 關閉查詢模式

更多指令 : https://docs.coreprotect.net/commands/

需要特別注意，在本服方塊紀錄只會保存 <span class="label label-info">30 天</span> ，
一旦一項方塊紀錄超過 <span class="label label-info">30 天</span> ，將會在隔日的 <span class="label label-info">00:00</span> 整自動執行方塊紀錄資料清除。

# 領地設置

           1. 手持木鋤

           2. 點按右鍵[基岩版長按]標定第一個座標 

           3. 點按左鍵[基岩版點按]標定第二個座標 

           4. 輸入 /res create <領地名> 即可建立領地。

在本服，一般玩家最多可以同時擁有 <span class="label label-info">15</span> 個領地，
每個領地的大小最大為 <span class="label label-info">160 × 160</span> （100 個區塊）大，
透過 <a href="https://www.brilliantw.net/%E8%B4%8A%E5%8A%A9%E8%B3%87%E8%A8%8A/#%E8%B4%8A%E5%8A%A9%E7%A6%8F%E5%88%A9">贊助伺服器</a> 可增加可同時擁有的領地數量與大小，
在框選領地時，無論如何，伺服器都會幫您自動框選 <span class="label label-info">-64 ~ 320</span> 的高度範圍，
故您不必特別框選高度（子領地需要）。

最後，設置領地是 <span class="label label-info">完全免費</span> 的，請善用領地系統保護個人資產防犯小人。

# 重啟回傳

本服為確保伺服器運作穩定會於每日的 <span class="label label-info">05:00</span> 進行一次伺服器重啟，
而在重啟前 <span class="label label-info">60、50、40、30、20、10、5</span> 秒，
伺服器會有聊天欄與語音通知，提醒玩家，
而透過我們自開發的系統，伺服器重啟時，系統會將您傳送至重啟分流等待伺服器重啟完畢，
伺服器重啟完畢後，系統會再將您自動傳回伺服器，全程不必手動操作。

# 礦物代換

在本服，礦物將不依照原版方式生成，透過此功能，即便玩家使用 X-Ray 透視到礦物，

可以最簡單的理解為：您挖石頭（深板岩）就是在抽獎，而抽獎獎勵是礦物。

而 <span class="label label-info">「替換」</span> 的標準為，一個礦物六面都是 <span class="label label-info">「石頭」</span> 
（負座標則為 <span class="label label-info">「深板岩」</span> ）。

# 圖繪使用
在本服，你可以透過指令生成地圖繪，但是有一些注意事項 ...

## 生成
透過 <span class="label label-info">/iframe create <地圖繪名稱> <圖片網址> <展示框行數（寬）> <展示框列數（高）></span> 指令格式，
你可以任意生成地圖繪，但是每生成 <span class="label label-info">一張地圖繪</span> ，就會消耗 <span class="label label-info">一張空白地圖</span> ，這點須特別注意。

![地圖繪生成](img/指令教學/圖繪使用/生成.gif)

## 刪除
透過 <span class="label label-info">/iframe delete <地圖繪名稱></span> 指令格式，
你可以將自己生成的地圖繪刪除掉，而被刪除的地圖繪，
倘若放置在展示框上，將會變成 <span class="label label-info">全白地圖</span> ，
倘若為物品型態，將會轉變回 <span class="label label-info">空白地圖</span> 。

![地圖繪刪除](img/指令教學/圖繪使用/刪除.gif)

## 改名
透過 <span class="label label-info">/iframe rename <舊地圖繪名稱> <新地圖繪名稱></span> 指令格式，
你可以將自己生成的地圖繪更改其地圖繪名稱。

![地圖繪改名](img/指令教學/圖繪使用/改名.gif)

## 資訊
透過 <span class="label label-info">/iframe info <地圖繪名稱></span> 指令格式，
或者手持地圖繪後輸入 <span class="label label-info">/iframe info</span> ，
即可查看該地圖繪的各項資訊，
包括但不限於： <span class="label label-info">建立者</span> 、 <span class="label label-info">建立時間</span> 、 <span class="label label-info">所使用的網址</span> 。

![地圖繪資訊](img/指令教學/圖繪使用/資訊.png)

## 取得
透過 <span class="label label-info">/iframe get <地圖繪名稱></span> 指令格式，
你可以取得自己曾經生成的地圖繪，
不需要再生成一個，但是每生成 <span class="label label-info">一張地圖繪</span> ，就會消耗 <span class="label label-info">一張空白地圖</span> ，這點須特別注意。。

![地圖繪取得](img/指令教學/圖繪使用/取得.gif)

## 複製
透過 <span class="label label-info">/iframe clone <被複製的地圖繪名稱> <複製的地圖繪名稱></span> 指令格式，
你可以再次生成一樣圖案與大小的地圖繪，但名稱與 <span class="label label-info">被複製的地圖繪</span> 需不同，
，但是每生成 <span class="label label-info">一張地圖繪</span> ，就會消耗 <span class="label label-info">一張空白地圖</span> ，這點須特別注意。。

![地圖繪複製](img/指令教學/圖繪使用/複製.gif)

## 標定
透過 <span class="label label-info">/iframe select</span> 來快速放置地圖畫了！
透過標定功能，玩家可以選擇要放置的展示框，
隨後再透過 <span class="label label-info">create</span> 、 <span class="label label-info">clone</span> 、 <span class="label label-info">get</span> 指令，即可直接放置。
（倘若是 <span class="label label-info">clone</span> 、 <span class="label label-info">get</span> 指令，標定大小需與展示框標定大小一致）

而在標定模式下， <span class="label label-info">create</span> 、 <span class="label label-info">clone</span> 、 <span class="label label-info">get</span> 指令
需使用與正常模式下不同的指令格式，才能夠被使用，如下：
### 1. create 指令格式：
1. 先輸入 <span class="label label-info">/iframe select</span> 標定展示框位置。
2. 使用 <span class="label label-info">/iframe create <地圖繪名稱> <圖片網址> selection</span> 來放置地圖繪。

### 2. clone 指令格式：
1. 先輸入 <span class="label label-info">/iframe select</span> 標定展示框位置。
2. 使用 <span class="label label-info">/iframe clone <被複製的地圖繪名稱> <複製的地圖繪名稱> selection</span> 來放置地圖繪。

### 3. get 指令格式：
1. 先輸入 <span class="label label-info">/iframe select</span> 標定展示框位置。
2. 使用 <span class="label label-info">/iframe get <地圖繪名稱> selection</span> 來放置地圖繪。

![地圖繪標定](img/指令教學/圖繪使用/標定.gif)

## 關閉

## 注意事項
1. 請確保你的背包空間是足夠的，若空間不足，會直接掉落於地上。
2. 請勿將包括但不限於 <span class="label label-info">色情、裸露、血腥</span> 之原圖檔轉換為地圖檔，
　違者將依服規處分。
3. 將原圖檔轉換為地圖檔前，請先經由原作者同意，再進行轉換使用，
　這是對每一位作者的基本尊重！
4. 生成的地圖畫為 2D 方式生成，故在色澤上可能會與原圖檔有所偏差。

## 領地權限相關
欲讓其他人在自己的領地內 <span class="label label-info">放上、拿下、轉動</span> 展示框的話，
該玩家需要擁有 <span class="label label-info">破壞（destroy）</span> 與 <span class="label label-info">使用容器（container）</span> 的領地權限。

- 由生怪磚生出的怪物，更改為在生出 <span class="label label-info">20秒</span> 後自動清除。

# 汙染值

 <span class="label label-info">聖域（終界）</span> 與 <span class="label label-info">深淵（地獄）</span> 的強烈排斥性，讓陌生人們無法自在的在這裡面行動，污染過多便會障目、疲憊、噁心甚至失去移動能力，直至死亡。
透過以下行為會造成 <span class="label label-info">汙染值</span> 的增加：

## 每時任務
 <span class="label label-info">每時任務</span> 分為 <span class="label label-info">I</span> 、 <span class="label label-info">II</span> 、 <span class="label label-info">III</span> 三個等級，
