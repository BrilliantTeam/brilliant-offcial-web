# 教學-進階查詢

           1. 輸入 /co l <參數> <頁數> 使用進階方塊查詢模式

           2. 輸入完畢後即可得知你所需要的一切資訊

進階方塊查詢的「參數種類」

<br />

` u:(玩家名) ` - 只限定這個玩家顯示在查詢紀錄中

   ↳ 例：` u:Meow_Rice_TW `   (查詢有關` Meow_Rice_TW `的紀錄)
   
<br />

` t:(時間) ` - 調整查詢紀錄所查詢的時間範圍

   ↳ 共支持查詢` 周、天、時、分、秒 `
      ↳ ( w , d , h , m , s )

           ↳ 例： t:2d   (查詢 2天前 的紀錄) 

           ↳ 例： t:3d2h   (查詢 3天又2小時前 的紀錄)

           ↳ 例： t:4.5d   (查詢 4.5天前 的紀錄)

` r:(半徑) ` - 只限定這個範圍會顯示在查詢紀錄中

   ↳ 例：` r:5 `   (查詢` 半徑5格內 `的紀錄)
   
<br />

` a:(動作) ` - 只限定這個動作會顯示在查詢紀錄中

   ↳ 共支持查詢以下項目

           ↳ block   ( 放置方塊 與 破壞方塊 )

           ↳ +block   ( 放置 方塊)

           ↳ -block   ( 破壞 方塊)

           ↳ click   (玩家 點擊 )

           ↳ container    (物品 拿出 或 放入 箱子等容器)

           ↳ +container    (物品 放入 箱子等容器)

           ↳ -container    (物品 拿出 箱子等容器)

           ↳ kill   (被殺死的 怪物和動物 )

           ↳ chat   (聊天欄中的 訊息 )

           ↳ command   (使用過的 指令 )

           ↳ session   (玩家 登入 和 登出 )

           ↳ +session   (玩家 登入 )

           ↳ -session   (玩家 登出 )

   ↳ 例：` a:+block `   (查詢` 放置方塊 `這個動作)

<br />

` b:(命名空間ID) ` - 只限定這個方塊會顯示在查詢紀錄

   ↳ 例：` b:shulker_box `   (紀錄中只會顯示有關` 原色界伏盒 `的紀錄)

<br />

` e:(命名空間ID) ` - 讓這個方塊不會出現在查詢紀錄中

   ↳ 例：` b:stone `   (紀錄中不會顯示有關` 石頭 `的紀錄)

<br />

進階方塊查詢的「實際應用」

` /co l u:Senz0309 t:3m r:20 a:+block b:stone `

查詢` Senz0309 `在` 3分鐘前 `於附近` 20格的範圍中 ` ` 放置 ` ` 石頭 ` 的紀錄

           ↳ 其中 Senz0309 就是 u: 的應用

           ↳ 其中 3分鐘前 就是 t: 的應用

           ↳ 其中 20格的範圍中 就是 r: 的應用

           ↳ 其中 放置 就是 a: 的應用

           ↳ 其中 石頭 就是 b: 的應用

# 教學-圖畫生成

           1. /mapview create url <展示框行數>:<展示框列數> <圖片網址>

` 電腦用戶 `如何獲取` 圖片網址 `:

           1. 選擇一張 圖片 

           2. 上傳至 Discord 後 點開圖片 

           3. 開啟圖片左下角 原尺寸圖片 `

           4. 複製 網頁連結 

           5. 這樣你就成功獲取 圖片網址 了

` 手機用戶 `如何獲取` 圖片網址 `:

           1. 選擇一張 圖片 

           2. 上傳至 Discord 後 點開圖片 

           3. 畫面右上角選擇 分享 

           4. 選擇 Discord 並複製 顯示出來的連結 

           5. 這樣你就成功獲取 圖片網址 了

注意事項

        1. 請確保你的背包空間是足夠的，若空間不足，會直接掉落於地上

        2. 你可以透過` 物品署名 `的功能來標記此地圖檔是屬於你的

        3. 請勿將` 色情、裸露、血腥 `之原圖檔轉換為地圖檔，違者將被處分 (服規2)

        4. 將原圖檔轉換為地圖檔前，請先經由原作者同意，再進行轉換使用

           - 這是對每一位作者的基本尊重！
