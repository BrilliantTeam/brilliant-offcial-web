# 方塊查詢

           1. 輸入 /co i 開啟查詢模式

           2. 點按您想查詢的方塊，會跳出系統訊息，可以看到 時間/玩家/行為

           3. 再次輸入 /co i 關閉查詢模式

更多指令 : https://docs.coreprotect.net/commands/

需要特別注意，在本服方塊紀錄只會保存` 30 天 `，
一旦一項方塊紀錄超過` 30 天 `，將會在隔日的` 00:00 `整自動執行方塊紀錄資料清除。

# 進階查詢

           2. 輸入完畢後即可得知你所需要的一切資訊

# 領地設置

           1. 手持木鋤

           2. 點按右鍵[基岩版長按]標定第一個座標 

           3. 點按左鍵[基岩版點按]標定第二個座標 

           4. 輸入 /res create <領地名> 即可建立領地。

在本服，每位玩家最多可以擁有` 50 `個領地，
每個領地的大小最大為` 160 × 160 `（100 個區塊）大，
在框選領地時，無論如何，伺服器都會幫你自動框選` -64 ~ 320 `的高度範圍，
故您不必特別框選高度（子領地需要）。

最後，設置領地是` 完全免費 `的，請善用領地系統保護個人資產防犯小人。

           1. 在您要上鎖的物品中放上告示牌。

           2. 在告示牌第一行寫上 [私人] 。

           3. 保存。這時您的 ID 將會自動出現在告示牌上。

              └ 若您要給好友開啟權限，請在第 3 、 4 行輸入好友遊戲 ID 即可。

> 

           1. 在告示牌上寫上 [紅石] 標籤，將允許紅石訊號的傳遞。
           
           2. 在已鎖上的方塊上放置第二個告示牌，並在告示牌上寫上 [共用] 標籤及玩家 ID，將允許其他玩家使用。
           
           3. 在告示牌上寫上 [所有人] 標籤，將允許全部玩家使用。
           
           4. 在告示牌上寫上 [計時:(秒數)] 標籤，將會固定在(秒數)秒後給該方塊輸出紅石訊號。

# 資源再生

在本服，我們使用自開發的[NatureRevive](https://forum.gamer.com.tw/Co.php?bsn=18673&sn=1037065)插件，來達成無須資源世界，但資源永不枯竭的自動化再生系統。

## 具體機制

當該區域有玩家活動過後，將會標記為` 「過期區塊」 `
當` 30天 `內其他玩家` 「沒有進行活動」 `時，將會` 「準備再生」 `。
` 「進行活動」 `包括：
- BlockBreakEvent（破壞方塊）
- BlockBurnEvent（方塊燃燒）
- BlockCookEvent（使用熔爐、煙燻爐、高爐 熔煉或烹飪）
- LeavesDecayEvent（樹葉腐爛）
- EntityDeathEvent（實體死亡）（特定條件：玩家為擊殺者，且被擊殺者不是玩家）
- BrewEvent（使用釀造台）
- FurnaceBurnEvent（使用熔爐、煙燻爐、高爐 熔煉或烹飪）
- EntityExplodeEvent（實體爆炸）
- BlockExplodeEvent（方塊爆炸）
- BlockPlaceEvent（放置方塊）

而當一個區塊被標記為` 「準備再生」 `後，
當有玩家載入到這些` 「準備再生」 `的區塊後，伺服器將` 「執行再生」 `，
屆時，這個區塊將會被重置回原本的樣子（除了礦物、結構、戰利品隨機化）。

## 例外情況

有設置領地的區塊將永遠不會被標記為` 「過期區塊」 `，
在本服設置領地是` 完全免費 `的，請善用領地系統保護個人資產防犯小人。

# 礦物代換

在本服，礦物將不依照原版方式生成，透過此功能，即便玩家使用 X-Ray 透視到礦物，
而嘗試挖掘這些礦物時，伺服器會將原有的礦物移除，並代換為石頭（深板岩）或者礦物。

可以最簡單的理解為：你挖石頭（深板岩）就是在抽獎，而抽獎獎勵是礦物。

## 何時代換

僅有` 「自然生成」 `之礦物會被此插件的功能` 「替換」 `，
而` 「替換」 `的標準為，一個礦物六面都是` 「石頭」 `
（負座標則為` 「深板岩」 `）。
