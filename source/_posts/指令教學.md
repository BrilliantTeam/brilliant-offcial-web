# 2. 領地指令

           1. 手持木鋤

           2. 點按右鍵[基岩版長按]標定第一個座標 

           3. 點按左鍵[基岩版點按]標定第二個座標 

           4. 輸入 /res create <領地名> 即可創建領地。

常用指令 : 

           1. /res rename <舊領地名稱> <新領地名稱> - 重新命名領地

           2. /res remove <領地名稱> - 刪除領地

           3. /res tpset – 將你所站的位置設為領地傳送點

           4. /res set <權限名稱> <false | true> - 設定領地權限

           5. /res padd <玩家名> - 給予玩家所有權限

           6. /res flags – 打開權限列表

           7. /res pset <玩家名稱> <權限名稱> <false | true> - 設置個別玩家權限

           8. /res tp <領地名稱> - 傳送到該領地

           9. /res expand <格數> - 向玩家所面對的方向往前擴大領地格數

           10. /res contract <格數> - 向玩家所面對的方向縮小領地格數

           11. /res subzone <領地名稱> <子區域名稱> - 在領地內創建子區域

           12. /res show – 顯示領地邊界

更多指令 : https://www.zrips.net/residence/commands

教學影片 : https://youtu.be/KwcPktN1JRs

# 5. 進階查詢

           1. 輸入 /co l <參數> 使用進階方塊查詢模式

           2. 輸入完畢後即可得知你所需要的一切資訊

進階方塊查詢的「參數種類」

<br />

` u:(玩家名) ` - 只限定這個玩家顯示在查詢紀錄中

   └ 例：` u:Meow_Rice_TW `   (查詢有關` Meow_Rice_TW `的紀錄)
   
<br />

` t:(時間) ` - 調整查詢紀錄所查詢的時間範圍

   └ 共支持查詢` 周、天、時、分、秒 `
      └ ( w , d , h , m , s )

           ├ 例： t:2d (查詢 2天前 的紀錄) 

           ├ 例： t:3d2h (查詢 3天又2小時前 的紀錄)

           └ 例： t:4.5d (查詢 4.5天前 的紀錄)

` r:(半徑) ` - 只限定這個範圍會顯示在查詢紀錄中

   └ 例：` r:5 ` (查詢` 半徑5格內 `的紀錄)
   
<br />

` a:(動作) ` - 只限定這個動作會顯示在查詢紀錄中

   └ 共支持查詢以下項目

           ├ block (放置方塊 與 破壞方塊)

           ├ +block (放置 方塊)

           ├ -block (破壞 方塊)

           ├ click (玩家 點擊)

           ├ container (物品 拿出 或 放入 箱子等容器)

           ├ +container (物品 放入 箱子等容器)

           ├ -container (物品 拿出 箱子等容器)

           ├ kill (被殺死的 怪物和動物)

           ├ chat (聊天欄中的 訊息)

           ├ command (使用過的 指令)

           ├ session (玩家 登入 和 登出)

           ├ +session (玩家 登入)

           └ -session (玩家 登出)

   └ 例：` a:+block ` (查詢` 放置方塊 `這個動作)

<br />

` b:(命名空間ID) ` - 只限定這個方塊會顯示在查詢紀錄

   └ 例：` b:shulker_box ` (紀錄中只會顯示有關` 原色界伏盒 `的紀錄)

<br />

` e:(命名空間ID) ` - 讓這個方塊不會出現在查詢紀錄中

   └ 例：` b:stone ` (紀錄中不會顯示有關` 石頭 `的紀錄)

<br />

進階方塊查詢的「實際應用」

` /co l u:Senz0309 t:3m r:20 a:+block b:stone `

查詢` Senz0309 `在` 3分鐘前 `於附近` 20格的範圍中 ` ` 放置 ` ` 石頭 ` 的紀錄

           ├ 其中 Senz0309 就是 u: 的應用

           ├ 其中 3分鐘前 就是 t: 的應用

           ├ 其中 20格的範圍中 就是 r: 的應用

           ├ 其中 放置 就是 a: 的應用

           └ 其中 石頭 就是 b: 的應用

# 6. 圖片指令

           1. /mapview create url <展示框行數>:<展示框列數> <圖片網址> - 即可獲取地圖畫
           
如何獲得圖片網址

           1. Discord中 私訊 胡桃BOT 任意圖片 即可獲得圖片網址
              └ 請注意！你需要開啟私人訊息權限，才可以私訊機器人！

注意事項

           1. 請確保你的背包空間是足夠的，若空間不足，會直接掉落於地上

           2. 你可以透過` 物品署名 `的功能來標記此地圖檔是屬於你的

           3. 請勿將` 色情、裸露、血腥 `之原圖檔轉換為地圖檔，違者將被處分 (服規2)

           4. 將原圖檔轉換為地圖檔前，請先經由原作者同意，再進行轉換使用

              └ 這是對每一位作者的基本尊重！

# 25. 展示語法
請注意！這些語法不用使用 / (斜線)，取而代之的是 [] 這兩個符號！

           1. [i] - 在聊天欄展示自己手中的物品

           2. [inv] - 在聊天欄展示自己的背包

           3. [e] - 在聊天欄展示自己的終界箱

           4. [任意指令] - 在聊天欄發送可以點擊的提示訊息（基岩版無法點擊）
              └ 例： 輸入[/spawn]就會在聊天欄顯示可以點擊的/spawn指令
                     點擊後會直接在點擊者的聊天欄填入此段指令（在提示新手指令用法上不錯用）
